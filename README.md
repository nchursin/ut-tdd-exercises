# Упражнения для тренинга "Юнит-тестирование + TDD"

В этом репозитории лежат упраженения для тренинга по юнит-тестирования.

1. Каждое упражнение надо открывать как отдельный проект в IDE.
1. Инструкции по настройке каждого упражнения одинаковые, но на всякий случай лежат в README.md соответствующей папки

Чтобы убедиться, что всё работает зайдите в папку `dices` и настройте проект по [инструкции](/dices/README.md).
