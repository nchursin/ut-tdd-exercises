# Roll Dice Game

## Практика String Calculator
Реализовать String Calculator по TDD (разработка через тестирование). Тесты должны быть понятными и читаемыми.

### Правила строкового калькулятора (Sting Calculator)

Не читайте следующее правило, пока не реализуете предыдущее.
#### Первый шаг. Сложение двух чисел
Написать функцию сложения чисел
```Python
def add(number: str) -> str:
```
Может принимать 0, 1 или 2 числа, разделенные запятой, и возвращать их сумму. Пустая строка вернет "0". Примеры: 
- "" должно возвращать "0" 
- "1" должно возвращать "1"
- "1.1,2.2" должно возвращать "3.3"
#### Второй шаг. Множество чисел
Функция add должна обрабатывать какое угодно количество чисел при сложении
#### Третий шаг. Новая строка как разделитель
- "1\n2,3" должно возвращать "6"
- "175.2,\n35" недопустимо и должно вернуть "Ожидается число, но в позиции 6 найдено '\n'"

#### Четвёртый шаг. Отсутствие числа в последней позиции
Не допускайте, чтобы ввод заканчивался разделителем: "1,3," недопустимо и должно возвращать сообщение "Ожидается число, но найдено EOF"

#### Пятый шаг. Пользовательские разделители
//[delimiter]\n[numbers]

Примеры:
- "//;\n1;2" должно возвращать "3"
- "//sep\n2sep3" должно возвращать "5"
- "//|\n1|2,3" недопустимо и должно вернуть "Ожидается '|', но в позиции 3 найдено ','"

#### Шестой шаг. Недопустимость отрицательных чисел
- "-1,2" недопустимо и должно вернуть "Отрицательные числа не разрешены: -1"
- "2,-4,-5" недопустимо и должно вернуть "Отрицательные числа не разрешены: -4, -5"


## Настройка окружения

### Требования

- python3 (3.10 - 3.14)

### Подготовка IDE

#### Для Mac

Если у вас OS X и вы собираетесь использовать VSCode, то надо знать, что стандартное расширение ms-python.python не
поддерживает встроенный интерпретатор.

**Если у вас python3 из любого другого источника (conda, brew, pyenv), то можно пропустить этот пункт**. Если нет, то
вот способ поставить.

```bash
# для OS X, если у вас установлен только встроенный интерпретатор
brew install pyenv
pyenv install 3.14
pyenv global 3.14
```

#### Для всех

```bash
cd new-ut-tdd/Python
python3 -m venv .venv
source .venv/bin/activate
cp .assets/pip.conf .venv/pip.conf # на случай, если у вас по умолчанию стоит рабочий реестр пакетов
pip install -r requirements.txt
pytest tests
```
