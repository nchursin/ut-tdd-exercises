# Практика 03

## Практика Mock
Использование тестовых дублеров - Mock.

**Задача:** Написать тесты на выигрыш/проигрыш с моками. [Код упражнения](/practice/day1/dices/)
- Игрок сделал правильную ставку и выиграл, т.е. у него вызвался Player.Take() для ставки и Player.Win() для победы
- Игрок сделал неправильную ставку и проиграл, т.е. у него вызвался Player.Take() для ставки и не вызвался Player.Win() для победы
- Задублируйте кубик
- Замокайте игрока
- Обращайте внимание на
   - Наименование теста
   - Структуру теста (AAA)
   - Лаконичность и выразительность теста
   - Понятность тестовых данных
